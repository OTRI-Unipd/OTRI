## config.json
In order to work properly most scripts require a `config.json` with the following structure:

```JSON
{
    "postgresql_username" : "",
    "postgresql_password" : "",
    "postgresql_host" : "",
    "postgresql_port" : "",
    "postgresql_database" : "",
    "alphavantage_api_key": ""
}
```
